

# INSPIRE OKC — World-Class Premium Makeover

## Vision
Transform INSPIRE OKC from a bold brutalist site into a **premium editorial experience** — think magazine-quality layouts, cinematic typography, smooth animations, and polished interactions — while preserving all your core logic, routes, and functionality.

---

## Phase 1: Design System & Foundation
- **Typography overhaul** — Large, confident editorial headings with elegant body text. Strong typographic hierarchy across all pages
- **Refined color system** — Keep black/white core but add subtle warm grays, refined accent tones, and sophisticated gradient overlays for depth
- **Spacing & rhythm** — Generous whitespace, consistent section padding, and a clear visual grid system
- **Component library upgrade** — Elevate buttons, cards, badges, and inputs with refined borders, subtle shadows, and smooth hover states

## Phase 2: Homepage — The Hero Experience
- **Cinematic hero reimagined** — Full-viewport hero with editorial-grade typography, parallax-style layering, and a striking call-to-action
- **Premium navigation bar** — Sleek fixed header with smooth scroll-linked opacity, refined mobile drawer with elegant transitions
- **Scroll progress indicator** — Refined, minimal progress bar
- **Section-by-section flow** — Each homepage section (Events, Blog, Podcast, Social, Graphics, TeeFee) redesigned as editorial "chapters" with full-width layouts, pull-quotes, and cinematic imagery treatment

## Phase 3: Content Pages Makeover
- **Story page** — Magazine-style article layout with large feature images, drop caps, and elegant reading experience
- **Podcast page** — Rich media player layout with episode cards, waveform visuals, and clean episode listing
- **Community programs** (Adulting, Singles, Workout, Self-Mastery) — Unified template with hero banners, program details cards, and clear CTAs
- **Apply page** — Polished multi-step or single-page form with clear progress and validation feedback
- **Info page** — Clean, organized information architecture with accordion sections or tabbed layout
- **Apps & Slides pages** — Grid-based gallery layouts with hover previews
- **Cartoonizer Lab** — Fun but polished interface for the creative tool

## Phase 4: Animations & Micro-Interactions
- **Page entrance animations** — Staggered fade-in and slide-up for content sections as they scroll into view
- **Smooth page transitions** — Elegant transitions between routes
- **Interactive hover effects** — Subtle scale, shadow lifts, and color shifts on cards and links
- **Scroll-triggered reveals** — Content sections animate in as the user scrolls, creating a storytelling feel
- **Button & link feedback** — Refined press states, underline animations, and loading indicators

## Phase 5: Supabase Backend Integration
- **Authentication** — Sign in/Sign up with email, proper session management, and auth state handling
- **User profiles** — Profile table linked to auth users (if needed for community features)
- **Admin panel** — Protected admin route with role-based access using a proper user_roles table
- **Content management** — Database-driven events, blog posts, podcast episodes, and community program data
- **Security** — RLS policies on all tables, security definer functions for role checks

## Phase 6: Mobile Excellence
- **Mobile-first refinement** — Every page optimized for mobile with touch-friendly interactions
- **Mobile navigation** — Elegant slide-out drawer menu with smooth animations
- **PWA install modal** — Polished install prompt with clear value proposition
- **Responsive imagery** — Properly sized images across all breakpoints

## Phase 7: Footer & Global Elements
- **Premium footer** — Multi-column layout with links, branding, social icons, and newsletter signup
- **Toast notifications** — Refined notification styling consistent with the editorial theme
- **404 page** — On-brand, helpful not-found page with navigation back
